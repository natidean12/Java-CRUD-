# Sistema de Gestão Acadêmica

Uma aplicação desktop desenvolvida em Java para gerenciar informações acadêmicas por meio de uma interface gráfica intuitiva. O sistema realiza operações completas de CRUD para Alunos e Professores, com armazenamento persistente de dados em MySQL utilizando JDBC.

# Funcionalidades
Gestão de alunos
Gestão de professores
Operações completas de CRUD
Interface desktop Java Swing
Integração em tempo real com MySQL
Validação de dados
Gerenciamento de transações (Commit e Rollback)
Validação de integridade referencial
Múltiplos relacionamentos (Muitos-para-Muitos)
Comunicação segura com o banco de dados usando PreparedStatement
